Movie API

This API allows you to receive information on movies, directors, and genres so that you can learn more about movies you have watched or are interested in. You are able to create a profile so you can save data about your favorite movies.

REQUEST URL HTTP METHOD Request Body Data Format Response Body Data Format
Return a list of ALL movies /movies GET None A JSON object holding data about all the movies
Return data about a SINGLE movie by description /movies/[description] GET None A JSON object holding data about a single movie, based on its description
Return data about a SINGLE movie by genre /movies/[genre] GET None A JSON object holding data about a single movie, based on the genre
Return data about a SINGLE movie by director name /movies/[director] GET None A JSON object holding data about a single movie, based on the director
Return data about a SINGLE movie by image URL /movies/[image] GET None A JSON object holding data about a single movie, based on image URL
Return data about a SINGLE movie by whether or not its featured /movies/[featured] GET None A JSON object holding data about a single movie, based on whether or not its featured
Return data about a genre (description) by name/title of movie /movies/[genre]/[name] GET None A JSON object holding data about the genre of a specific movie
Return data about a director (bio, birth year, death year) by name /movies/[director]/[name] GET None A JSON object holding data about a director, based on name
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added) /movies PUT None Text indicating that a movie has been added to a list
Remove a movie from their list of favorites /movies/[ID] DELETE None Text that a movie has been removed
Allow new users to register /users POST A JSON object holding data about the user to add Text indicating that user has been registered
Allow new users to update their user info (username) /users PUT None Text indicating that their username has been updated
Allow existing users to deregister (showing text that a user email has been removed) /users DELETE None Text that a user email has been removed
