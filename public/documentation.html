Movie API

This API allows you to receive information on movies, directors, and genres so that you can learn more about movies you have watched or are interested in. You are able to create a profile so you can save data about your favorite movies.

What kind of data does the endpoint return? Endpoint URL Query Parameters HTTP METHOD Request body data format Response body data format
Return a list of ALL movies to the user /movies None GET None A JSON object holding data about all the movies
Return ALL data (description, genre, director, image URL, whether or not it's featured) about a single movie by the user entering the TITLE of a movie /movies :title GET None A JSON object holding all data about a single movie: {Title: "Up", Genre: "Animation", Director: "Pete Docter", image URL: link here, Feature Status: yes}
Return data about a genre (description) by the user entering the name of the genre /movies/genre :genreName GET None A JSON object holding data regarding the genre of a single movie
Return data about a director by the user entering the NAME of the director /movies/directors :directorName GET None A JSON object holding all data about a single director: {Name:Pete Docter, Birthdate: 10/09/1968}
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added) /users/id :favorites PUT None Text indicating: "Movie has been added to Favorites List"
Remove a movie from their list of favorites /users/id :favorites DELETE None Text indicating: "Movie has been removed from Favorites List"
Allow new users to register /users :id POST A JSON object holding data about the user to add: {name: Olivia, email address: o123@gmail.com} Text indicating: "User has been registered"
Allow new users to update their user info (username) /users :id PUT None Text indicating: "User information has been updated"
Allow existing users to deregister (showing text that a user email has been removed) /users :id DELETE None Text indicating: "User email has been successfully unregistered"
